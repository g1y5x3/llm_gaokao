Response:

考虑方程：
\[ x^2 - 2x - 3 = 0 \]

我们可以使用因式分解法来解这个方程。首先，寻找两个数，它们的乘积为常数项 \(-3\)，并且它们的和为线性项的系数 \(-2\)。

这两个数是 \(-3\) 和 \(1\)，因为：
\[ (-3) \cdot 1 = -3 \]
\[ (-3) + 1 = -2 \]

因此，方程可以分解为：
\[ (x - 3)(x + 1) = 0 \]

解这个方程得到两个根：
\[ x - 3 = 0 \quad \text{或} \quad x + 1 = 0 \]
\[ x = 3 \quad \text{或} \quad x = -1 \]

现在我们知道二次函数 \( x^2 - 2x - 3 \) 在 \( x = -1 \) 和 \( x = 3 \) 处有零点。接下来，我们需要确定在这些零点之间和外部的函数值的符号。

1. \( x < -1 \)

3. \( x > 3 \)

- 在 \( x < -1 \) 区间内，选择 \( x = -2 \)：

  \[ 0^2 - 2(0) - 3 = -3 \]

- 在 \( x > 3 \) 区间内，选择 \( x = 4 \)：
  \[ 4^2 - 2(4) - 3 = 16 - 8 - 3 = 5 \]

因此，函数 \( x^2 - 2x - 3 \) 在区间 \( -1 < x < 3 \) 内为负值，而在区间 \( x < -1 \) 和 \( x > 3 \) 内为正值。

综上所述，不等式 \( x^2 - 2x - 3 < 0 \) 的解集为：
\[ -1 < x < 3 \]

\boxed{(-1, 3)}

\[
\frac{2}{z} = \frac{2}{1 + yi}
\]

\[
\frac{2}{1 + yi} \cdot \frac{1 - yi}{1 - yi} = \frac{2(1 - yi)}{(1 + yi)(1 - yi)} = \frac{2(1 - yi)}{1 + y^2}
\]
因此，
\[
\frac{2}{z} = \frac{2(1 - yi)}{1 + y^2} = \frac{2}{1 + y^2} - \frac{2yi}{1 + y^2}
\]

现在我们将 \( z \) 和 \( \frac{2}{z} \) 相加：
\[
z + \frac{2}{z} = (1 + yi) + \left( \frac{2}{1 + y^2} - \frac{2yi}{1 + y^2} \right)
\]
将实部和虚部分开：
\[
z + \frac{2}{z} = \left( 1 + \frac{2}{1 + y^2} \right) + \left( y - \frac{2y}{1 + y^2} \right)i
\]

由于 \( z + \frac{2}{z} = m \) 是实数，虚部必须为零：
\[
y - \frac{2y}{1 + y^2} = 0
\]
解这个方程：
\[
y \left( 1 - \frac{2}{1 + y^2} \right) = 0
\]
\[
y \left( \frac{1 + y^2 - 2}{1 + y^2} \right) = 0
\]

1. 当 \( y = 0 \) 时：
\[
z = 1
\]
\[
z + \frac{2}{z} = 1 + \frac{2}{1} = 3

\[

综上所述，实数 \( m \) 的可能值为 \( 3 \) 或 \( 2 \)。因此，实数 \( m \) 的值为：
